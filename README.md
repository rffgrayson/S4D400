# S4D400 - Basic ABAP Programming
My work from completing SAP's S4D400 course on basic ABAP programming.

## What I learned

- ABAP Development Tools (ADT) for Eclipse
- Basic ABAP language techniques
- Object-oriented programming basics
- ABAP SQL for database access
- Simple and complex data types
- Business objects and entity manipulation
- ABAP RESTful Application Programming Model (RAP)

## Tech stack

- SAP BTP ABAP Environment
- ABAP Development Tools (Eclipse)
- abapGit for version control
- ABAP RESTful Programming Model

## Setup

Clone with abapGit:
1. Install abapGit in your system
2. Create new online repo
3. Pull objects

## Notes

This is the foundation course for ABAP. Course content works on SAP S/4HANA 2020+ and BTP ABAP Environment.
